# streamlit-demo

streamlit+langchain+ChatGPT を使って ChatGPT クローンを作る
Web 要約とか Youtube の要約をする機能も追加する

## 環境

Windows11
Python 3.10.0

## 実行方法

streamlit のインストール

```
pip install streamlit
```

実行

```
streamlit run app.py
```
